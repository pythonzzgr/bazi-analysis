# **Phase 4: 신강/신약 판단 (Strength Analysis)**

일간이 주변 세력에 비해 얼마나 힘이 있는지를 판단하여 용신(用神) 선정의 근거를 마련합니다.

### **1\. 판단 3대 지표**

1. **득령(得令):** 월지가 일간을 돕는 오행인가? (비겁/인성)  
2. **득지(得地):** 일지에 일간의 뿌리(통근)가 있는가?  
3. **득세(得勢):** 월지 외 다른 간지들에 일간을 돕는 세력이 많은가?

### **2\. 알고리즘 로직**

* total\_score를 100점으로 환산했을 때, 일간을 돕는 인성(印星)과 비겁(比劫)의 점수 합계를 산출합니다.  
* **신강(Strong):** 합계 점수가 50\~55점 이상.  
* **신약(Weak):** 합계 점수가 45\~50점 이하.  
* **중화(Balanced):** 에너지가 균형을 이룬 상태.

### **3\. 출력 데이터**

{  
  "strength\_status": "身强(Strong)",  
  "analysis": {  
    "self\_support\_score": 65.0,  
    "is\_deuk\_ryeong": true,  
    "is\_deuk\_ji": true,  
    "is\_deuk\_se": false  
  }  
}

**Reference:** 『적천수(適天髓)』 종격 및 신강신약론 참조.
